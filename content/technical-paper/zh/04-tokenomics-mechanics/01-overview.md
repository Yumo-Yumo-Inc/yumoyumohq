# 04 — 代币经济学机制

本节涵盖 Yumo Yumo 协议的经济机制：INT、bINT、ePoints 与 Foundation NFT 各自承担的角色；发行与转换生命周期；奖励上限；质押、金库与回购销毁机制；供应量分配；流通供应量预测；以及 INT 效用。

司法管辖区特定的法律意见由法律层管理。

## 4.0 机制接口

| 机制 | 公开范围 |
|---|---|
| INT | 主链上代币与流通会计 |
| bINT | 已验证贡献的预结算奖励表示 |
| ePoints | 应用内进度与季节性活动层 |
| Foundation NFT | 早期贡献与等级表示 |
| 质押 | 锁仓、累积与协议参与语义 |
| 金库 | 收益流入、营运、储备与代币经济之间的关系 |
| 供应量与分配 | 总供应量、轨道百分比、代币数量（4.16–4.18） |
| 发行与解锁 | 基于 MAU 的每日发行级距、轨道解锁时间表（4.19–4.21） |
| 奖励上限 | 按用户等级的每收据与每日上限（4.22–4.23） |
| 转换与流通 | bINT→INT 固定 1:1 比率、流通供应量模型（4.24–4.26） |
| INT 效用 | 奖励、质押、回购销毁、治理、绑定访问（4.27–4.29） |
