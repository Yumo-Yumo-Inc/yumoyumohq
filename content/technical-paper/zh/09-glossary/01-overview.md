# 09 — 詞彙

本詞彙彙整技術文件中使用的協議術語，並提供簡短且一致的定義。每個術語連回相關的機制、綱要或公式章節。

## 9.1 術語家族

| 術語家族 | 範圍 | 連結 |
|---|---|---|
| 代幣與鏈上資產 | INT、bINT、鑄造、銷毀、質押、金庫 | 04 |
| 經濟機制 | 發放、歸屬、回購銷毀、貢獻軌道 | 04 |
| 身分與貢獻 | 錢包、使用者、貢獻事件、支出證明 | 02、03 |
| 信任與品質 | 信任分數、健康分數、標準商品、k-匿名性 | 02、03、05 |
| 管線與基礎設施 | OCR、LLM、規則層、分類帳、Merkle 根 | 01、02、05 |
| 合規與法律 | KVKK、GDPR、MiCA、Howey、資料控制者 | 08 |
| 指標與產品術語 | MAU、收據、品項、審查佇列 | 02、03、04 |
