# Yumo Yumo Technical Whitepaper

Yumo Yumo 技術文件是協議的架構與機制規格說明。《願景文件》承載市場意圖與經濟敘事；本文件則解釋收據處理、信任層、獎勵會計、資料綱要與鏈上結算的運作方式。
