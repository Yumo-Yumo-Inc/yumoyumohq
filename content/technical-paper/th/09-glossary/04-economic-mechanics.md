# กลไกเศรษฐกิจ

## กลไกเศรษฐกิจ

- **TGE (Token Generation Event)** — ช่วงเวลาที่ INT กลายเป็นสินทรัพย์บนเชนที่ใช้งานได้ และ cPoints ถูกย้ายเป็น bINT
- **เส้นโค้งการปล่อย** — กำหนดการปล่อยรางวัลผู้ใช้ จัดทำดัชนีตามผู้ใช้งานประจำเดือนและเวลานับตั้งแต่ TGE *ดู: 04*
- **อัตราส่วนการแปลง (bINT → INT)** — ตัวคูณที่จัดทำดัชนีตามปีซึ่งกำหนดจำนวน INT ที่หน่วย bINT แปลงเป็น *ดู: 04*
- **เพดาน bINT รายวัน** — จำนวน bINT สูงสุดที่ผู้ใช้สามารถรับได้ต่อวัน คำนวณเป็น `base_cap × level_multiplier × health_score` *ดู: 03 ชั้นความน่าเชื่อถือ, 04 กลไก Tokenomics*
- **ซื้อคืนและเผา (BBB)** — กลไกการเผาตามกฎเกณฑ์ที่ใช้ภายใต้นโยบายคลังสำรองเพื่อลดการหมุนเวียนของ INT รายละเอียดการดำเนินการเป็นปฏิบัติการ *ดู: 04*
- **Liquidity Bootstrapping Pool (LBP)** — รูปแบบพูล DEX ที่ใช้เปิดตัวโทเค็นใหม่ด้วยน้ำหนักที่ปรับได้ พิจารณาสำหรับสภาพคล่อง INT เริ่มต้น
- **Single-sided LP** — การจัดหาสภาพคล่องที่ต้องการสินทรัพย์เพียงหนึ่งในสองของคู่ ใช้เพื่อเริ่มต้นพูล INT เริ่มต้นด้วย LP ที่ถูกล็อก
