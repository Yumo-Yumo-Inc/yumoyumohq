# โทเค็นและสินทรัพย์บนเชน

## โทเค็นและสินทรัพย์บนเชน

- **INT** — โทเค็นอรรถประโยชน์ที่โอนได้ของ Yumo Yumo ออกบน Solana เป็นโทเค็น SPL (ทศนิยม = 6 อุปทานทั้งหมด = 99,000,000,000) *ดู: 04 กลไก Tokenomics*
- **bINT** — เครดิตการมีส่วนร่วมแบบ soulbound ที่ mint ลงบน ATA ที่แช่แข็งของผู้ใช้เมื่อใบเสร็จได้รับการยืนยัน แปลงเป็น INT ตามอัตราส่วนที่จัดทำดัชนีตามปี *ดู: 04 กลไก Tokenomics*
- **ePoints** — เครดิตข้อมูลเชิงลึกแบบ soulbound ที่กำหนดมูลค่าเป็น USD สะท้อน "ต้นทุนที่ซ่อนอยู่" ที่เปิดเผยสำหรับครัวเรือนตามเวลา *ดู: 04 กลไก Tokenomics, 05 รูปแบบข้อมูลและ API*
- **cPoints** — คะแนนการมีส่วนร่วมก่อน TGE ย้ายเป็น bINT ที่ Token Generation Event และเก็บเป็นบันทึกคะแนนที่เก็บถาวร *ดู: 04 กลไก Tokenomics*
- **Foundation NFT (Yumbie)** — NFT NonTransferable แบบ Token-2022 ที่แทนอัตลักษณ์ Yumbie ของผู้ใช้ พัฒนาเป็น Smart Agent ที่ระดับ 30 *ดู: 00 ทีมและปรัชญาการสร้าง, 04 กลไก Tokenomics*
- **Smart Agent** — การวิวัฒนาการหลังระดับ 30 ของ Foundation NFT กิจกรรม mint ทิศทางเดียว *ดู: 04 กลไก Tokenomics*
- **SPL Token** — มาตรฐานโทเค็น Solana Program Library "ERC-20 ของ Solana" ใช้สำหรับ INT
- **Token-2022** — มาตรฐานโทเค็นของ Solana ที่เปิดใช้งานส่วนขยาย ใช้สำหรับ Foundation NFTs พร้อมส่วนขยาย NonTransferable
- **Frozen ATA** — Associated Token Account ที่ถือ bINT (หรือ ePoints) และผูกกับผู้ใช้โดย mint authority บังคับใช้พฤติกรรม soulbound ในชั้นโปรโตคอล *ดู: 04 กลไก Tokenomics*
- **Soulbound** — พฤติกรรมโทเค็นที่ผูกกับกระเป๋าเงินเดียวตลอดอายุของสินทรัพย์
