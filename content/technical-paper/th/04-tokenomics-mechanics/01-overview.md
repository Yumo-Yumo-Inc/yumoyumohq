# 04 — กลไก Tokenomics

ส่วนนี้ครอบคลุมกลไกทางเศรษฐศาสตร์ของโปรโตคอล Yumo Yumo: บทบาทของ INT, bINT, ePoints และ Foundation NFT; วงจรชีวิตการปล่อยและการแปลง; เพดานรางวัล; กลไก staking, คลังสำรอง (treasury) และการซื้อคืน/เผา (buy-back/burn); การจัดสรรอุปทาน; การคาดการณ์อุปทานหมุนเวียน; และอรรถประโยชน์ของ INT

ความเห็นทางกฎหมายเฉพาะเขตอำนาจศาลจัดการอยู่ในชั้นกฎหมาย

## 4.0 พื้นผิวกลไก

| กลไก | ขอบเขตสาธารณะ |
|---|---|
| INT | โทเค็นหลักบนบล็อกเชน และการบัญชีหมุนเวียน |
| bINT | ตัวแทนรางวัลก่อนการชำระรอบ สำหรับการมีส่วนร่วมที่ได้รับการยืนยัน |
| ePoints | ชั้นความก้าวหน้าในแอปพลิเคชันและกิจกรรมตามฤดูกาล |
| Foundation NFT | ตัวแทนการมีส่วนร่วมระยะแรกและระดับ |
| Staking | ความหมายของการล็อก การสะสม และการมีส่วนร่วมในโปรโตคอล |
| Treasury | ความสัมพันธ์ระหว่างกระแสรายได้ การดำเนินงาน สำรอง และเศรษฐกิจโทเค็น |
| อุปทานและการจัดสรร | อุปทานรวม สัดส่วนแต่ละราง จำนวนโทเค็น (4.16–4.18) |
| การปล่อยและการปลดล็อก | ช่วงการปล่อยรายวันตาม MAU กำหนดการปลดล็อกแต่ละราง (4.19–4.21) |
| เพดานรางวัล | เพดานต่อใบเสร็จและเพดานรายวันตามระดับผู้ใช้ (4.22–4.23) |
| การแปลงและอุปทานหมุนเวียน | อัตราส่วน bINT→INT คงที่ 1:1 โมเดลอุปทานหมุนเวียน (4.24–4.26) |
| อรรถประโยชน์ของ INT | รางวัล staking ซื้อคืนและเผา ธรรมาภิบาล การเข้าถึงแบบวางหลักประกัน (4.27–4.29) |
