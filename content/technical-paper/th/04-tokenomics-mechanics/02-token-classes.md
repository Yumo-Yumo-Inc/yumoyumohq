# ประเภทของสินทรัพย์

## 4.1 สินทรัพย์สี่ประเภท

Yumo Yumo ทำงานด้วยสินทรัพย์สี่ประเภท แต่ละประเภทมีบทบาทเฉพาะตัว มีเพียงสองประเภทเท่านั้นที่เป็นโทเค็นบนบล็อกเชน อีกสองประเภทเป็นบันทึกนอกบล็อกเชน

| ประเภท | รูปแบบ | โมเดลการโอน | บทบาท |
|---|---|---|---|
| **INT** | โทเค็น SPL บน Solana | โอนได้ในตลาด | การประสานงานระดับโปรโตคอล การ staking และแรงจูงใจของระบบนิเวศ พารามิเตอร์ด้านอุปทานระบุไว้ใน Vision Paper |
| **bINT** | หน่วยบัญชีนอกบล็อกเชน (ชั้นดำเนินงาน) | ชำระเป็น INT ผ่านวงจรชีวิตที่กำหนดไว้ | ชั้นบัญชีการมีส่วนร่วมระหว่างงานกับรางวัล |
| **ePoints** | บันทึกนอกบล็อกเชน อ้างอิงมูลค่าเป็น USD | เครดิตข้อมูลเชิงลึกภายในผลิตภัณฑ์ | บันทึกต้นทุนแฝงระดับครัวเรือนที่ตรวจพบจากใบเสร็จที่ได้รับการยืนยันแต่ละใบ |
| **Foundation NFT (Yumbie)** | สินทรัพย์ Token-2022 ที่โอนไม่ได้ | โอนไม่ได้ | อัตลักษณ์ถาวร พัฒนาเป็น Smart Agent ตามจุดไมล์ที่กำหนดไว้ใน Vision Paper |

### เหตุใดจึงมีสี่ประเภท

Vision Paper อธิบายเหตุผลจากมุมมองประสบการณ์ผู้ใช้ ส่วนเหตุผลเชิงกลไกคือการแยกหน้าที่รับผิดชอบ:

- INT เคลื่อนที่ผ่านตลาดและตลาดแลกเปลี่ยน เป็นสินทรัพย์ที่โอนได้และทดแทนกันได้
- bINT ใช้วัดการมีส่วนร่วมและชำระเป็น INT เป็นหน่วยนอกบล็อกเชน การบัญชีจึงพัฒนาได้โดยไม่ต้องย้ายระบบบนบล็อกเชน
- ePoints นำพาสัญญาณข้อมูลเชิงเศรษฐกิจในฐานะบันทึกนอกบล็อกเชนของตัวเอง การวิเคราะห์ผู้ใช้จึงเติบโตได้ในขณะที่อุปทานของ INT ยังคงตายตัว
- Foundation NFT นำพาความต่อเนื่องของอัตลักษณ์ในฐานะสินทรัพย์ Token-2022 ที่โอนไม่ได้ หนึ่งใบต่อหนึ่งกระเป๋าเงิน

## 4.2 โครงสร้างอำนาจ

อำนาจแตกต่างกันตามว่าสินทรัพย์ประเภทนั้นอยู่บนบล็อกเชนหรือนอกบล็อกเชน

- **อำนาจการ mint INT** — ถือไว้จนกว่าจะ mint อุปทานทั้งหมด ณ genesis เท่านั้น จากนั้นปิด หลัง genesis จะไม่สามารถ mint INT ได้อีก การแจกจ่ายเป็นการโอนจากคลังสำรองผ่านตัวกระจาย (distributor) ที่ผ่านการตรวจสอบ (4.15)
- **คลังสำรองและการเผา INT** — ถือโดย Squads multisig โดยแยกการอนุมัติสำหรับการลงนามรากการแจกจ่าย (distribution root) การเคลื่อนย้ายคลังสำรอง และการเรียกคืนสำรอง (clawback)
- **bINT และ ePoints** — หน่วยบัญชีนอกบล็อกเชนในชั้นดำเนินงาน ไม่มีอำนาจ mint หรือ freeze บนบล็อกเชน ยอดคงเหลือชำระเป็น INT ผ่านวงจรชีวิตใน 4.4
- **Foundation NFT** — Token-2022 พร้อมส่วนขยายโอนไม่ได้ (non-transferable) mint โดยแบ็กเอนด์หนึ่งครั้งต่อหนึ่งกระเป๋าเงิน การโอนไม่ได้ถูกบังคับใช้ที่ชั้นโปรแกรมโทเค็น

การเก็บ bINT และ ePoints ไว้นอกบล็อกเชนช่วยขจัดอำนาจบนบล็อกเชนต่อเหตุการณ์ออกจากเส้นทางการมีส่วนร่วม อำนาจระดับ INT เพียงอย่างเดียวที่คงอยู่หลัง genesis คือ multisig เหนือคลังสำรอง รากการแจกจ่าย และการเผา
