# พื้นผิวสัญญาอัจฉริยะ

## 4.15 โมเดลโปรแกรม

v1 ทำงานบนโปรแกรมที่ผ่านการตรวจสอบและใช้งานกันอย่างแพร่หลาย แทนที่จะใช้โค้ดโปรโตคอลแบบเฉพาะ ไม่มีการปรับใช้โปรแกรมบนบล็อกเชนแบบสั่งทำพิเศษสำหรับชั้นรางวัล ฟังก์ชันบนบล็อกเชนแต่ละอย่างแมปไปยังโปรแกรมที่ปรับใช้แล้วและผ่านการตรวจทานจากภายนอก

| ฟังก์ชัน | โปรแกรม | โมเดลอำนาจ |
|---|---|---|
| การออกและการเผา INT | SPL Token | ปิดอำนาจ mint หลัง genesis; การเผาอยู่ภายใต้ multisig |
| การแจกจ่ายและเคลมรางวัล | merkle distributor ที่ผ่านการตรวจสอบ | รากต่อรอบ (per-epoch root); อำนาจตั้งรากเป็น multisig |
| คลังสำรองและอำนาจ | Squads multisig | แยกการอนุมัติราก / คลังสำรอง / clawback |
| Foundation NFT | Token-2022 (NonTransferable) | แบ็กเอนด์ mint หนึ่งใบต่อหนึ่งกระเป๋าเงิน |
| คำมั่นสัญญาความโปร่งใส | Memo program | รากของรอบและแฮชชุดข้อมูลถูกเขียนบนบล็อกเชน |

bINT และ ePoints เป็นหน่วยบัญชีนอกบล็อกเชน ไม่ใช่โทเค็นบนบล็อกเชน ยอดคงเหลืออยู่ในชั้นดำเนินงานและชำระเป็น INT ผ่านตัวกระจาย (distributor)

## 4.16 สิ่งที่ขึ้นบนเชน

ชั้นบนบล็อกเชนนำพาเหตุการณ์โทเค็น INT รากการแจกจ่ายต่อรอบ การเปลี่ยนแปลงอำนาจคลังสำรอง และคำมั่นสัญญาความโปร่งใส ชั้นนอกบล็อกเชนนำพาเครื่องยนต์รางวัล (การมีส่วนร่วม → จำนวน → ราก) เนื้อหาใบเสร็จ สัญญาณความน่าเชื่อถือ และประวัติพฤติกรรม

ชุดข้อมูลรางวัลที่เผยแพร่ถูกเขียนลงในที่จัดเก็บถาวร แฮชของชุดข้อมูลและรากของรอบถูกยืนยัน (commit) บนบล็อกเชน ฝ่ายภายนอกสามารถคำนวณยอดคงเหลือของตนใหม่จากข้อมูลที่เผยแพร่และตรวจทานกับรากบนบล็อกเชนได้ ในขณะที่เนื้อหาใบเสร็จยังคงอยู่ในชั้นข้อมูลนอกบล็อกเชน

## 4.17 ความสมบูรณ์ของการชำระ

ก่อนที่รากการแจกจ่ายจะถูกลงนาม ผู้ตรวจสอบอิสระจะคำนวณรากนั้นใหม่จากบัญชีแยกประเภทแหล่งเดียวกันและตรวจทานค่าคงที่ของการจัดสรรสะสม (invariants) (4.18) รากที่ไม่ตรงกับการคำนวณใหม่ หรือที่จะละเมิดเพดานการจัดสรร จะไม่ดำเนินการสู่การลงนาม การลงนามรากและการเคลื่อนย้ายคลังสำรองต้องใช้การอนุมัติหลายฝ่ายผ่าน Squads multisig

การแยกส่วนนี้ทำให้การคำนวณรางวัล การตรวจสอบอิสระ และการเคลื่อนย้ายเงินทุนอยู่ในมือที่แตกต่างกัน: เซิร์ฟเวอร์ที่ถูกบุกรุกเพียงเครื่องเดียวไม่สามารถเคลื่อนย้ายเงินทุนได้ด้วยตัวเอง

## 4.18 ท่าทีการตรวจสอบ

พื้นผิวบนบล็อกเชนพึ่งพาโปรแกรมที่ผ่านการตรวจสอบแล้วและใช้งานในการผลิตอย่างกว้างขวาง เครื่องยนต์รางวัลนอกบล็อกเชนและผู้ตรวจสอบอิสระจะถูกตรวจทานก่อนการเปิดตัว พร้อมคลังรายงานสาธารณะและช่องทางรายงานความปลอดภัย ขอบเขตและลิงก์รายงานจะถูกเผยแพร่เมื่อการตรวจทานเสร็จสมบูรณ์
