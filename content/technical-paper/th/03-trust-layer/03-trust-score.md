# คะแนนความน่าเชื่อถือ

## 3.3 คะแนนความน่าเชื่อถือต่อใบเสร็จ

คะแนนความน่าเชื่อถือสำหรับใบเสร็จเป็นตัวเลขในช่วง `[0, 1]` ที่ได้จากการรวมสัญญาณแบบถ่วงน้ำหนัก ยิ่งค่าสูงยิ่งดี คะแนนนี้จะถูกบันทึกลงในระเบียนใบเสร็จ (ดู 05 §5.3) พร้อมรายการสัญญาณที่มีผลต่อการตัดสินใจ

### ตระกูลสัญญาณ

คะแนนอาศัยตระกูลสัญญาณสี่ตระกูล แต่ละตระกูลผลิตสัญญาณเดี่ยวหนึ่งตัวหรือมากกว่า ชั้นนี้จะประกอบสัญญาณเหล่านั้นเข้าเป็นคะแนนสุดท้าย

| ตระกูล | สิ่งที่สังเกต |
|---|---|
| **ความมั่นใจของไปป์ไลน์** | ไปป์ไลน์ต้นทางมีความมั่นใจเพียงใดในการแยกข้อมูล (ความมั่นใจ OCR ความมั่นใจ LLM การปรับให้ตรงกันของชั้นกฎ) |
| **ความสอดคล้องของผู้ค้า** | ผู้ค้า สาขา และเทมเพลตใบเสร็จตรงกับสิ่งที่เราเคยเห็นจากผู้ค้ารายนี้ก่อนหน้านี้หรือไม่ |
| **ความสมเหตุสมผลทางเวลา** | วันที่ เวลา และรูปแบบการอัปโหลดของใบเสร็จสอดคล้องกับพฤติกรรมปกติหรือไม่ |
| **ประวัติผู้ใช้** | คุณภาพการมีส่วนร่วมล่าสุดของผู้ใช้ ภายในหน้าต่างเลื่อน |

ภายในแต่ละตระกูล สัญญาณเดี่ยวจะถูกบันทึกพร้อมค่าที่สังเกตได้และธงการมีส่วนร่วม (`signal_used` / `signal_skipped`) น้ำหนักที่แน่นอน เกณฑ์ตระกูล และกฎการข้ามได้รับการจัดการในชั้นปฏิบัติการภายใน

### วงคะแนน

คะแนนสุดท้ายจะถูกจัดกลุ่มเป็นวงสำหรับการใช้งานต่อไปนี้:

- **สูง** — ใบเสร็จผ่านเพื่อรับเครดิต bINT เต็ม
- **ปานกลาง** — ใบเสร็จผ่านเพื่อรับเครดิต bINT ที่ลดลง ผู้ใช้จะเห็นตัวอย่างที่ตรวจสอบแล้วและจำนวน bINT โดยไม่ต้องผ่านขั้นตอนเพิ่มเติม
- **ต่ำ** — ใบเสร็จถูกระงับเพื่อตรวจสอบ (ดู 3.7) ผู้ใช้ได้รับแจ้งว่ากำลังตรวจสอบอยู่
- **ปฏิเสธ** — ใบเสร็จเข้าสู่สถานะปฏิเสธ ผู้ใช้เห็นหมวดหมู่เหตุผลในรูปแบบภาษาง่าย

ขอบเขตของวงจะได้รับการปรับเทียบเป็นระยะตามผลลัพธ์ที่สังเกตได้ และได้รับการจัดการในชั้นปฏิบัติการภายใน

## 3.4 สิ่งที่บันทึกใบเสร็จบรรจุไว้

บล็อกความน่าเชื่อถือของใบเสร็จจะแสดงตระกูลสัญญาณที่มีผลต่อการตัดสินใจและช่วงคะแนนที่ได้รับ ส่วนค่าสัญญาณรายตัว น้ำหนัก และคะแนนโดยละเอียดจะอยู่ในการกำหนดค่าความน่าเชื่อถือภายใน พื้นผิวที่มุ่งสู่ผู้ใช้จะสื่อสาร **ผลลัพธ์** (จำนวน bINT การระงับ หรือการปฏิเสธ) และ **หมวดหมู่ของเหตุผล** เมื่อจำเป็น

นี่เป็นการออกแบบโดยตั้งใจ ช่วงคะแนนสาธารณะช่วยให้ผลลัพธ์ชัดเจนต่อผู้ใช้ ขณะที่คะแนนภายในยังคงรองรับการปรับเทียบของระบบ
