# แผนงาน SDK

## 5.14 แผนงาน SDK

| ไตรมาสเชิงบ่งชี้ | SDK |
|---|---|
| Q+1 | TypeScript SDK (public API) |
| Q+1 | Solana program TypeScript bindings |
| Q+2 | Swift / Kotlin mobile SDK |
| Q+3 | Python SDK สำหรับ B2B |
| Q+4 | R / Stata SDK สำหรับงานวิจัย |

B2B Python SDK เป็นพื้นผิวนักพัฒนาที่มีเลเวอเรจสูงที่สุด — ทีมข้อมูลของผู้ซื้อ B2B ส่วนใหญ่ใช้ Python เป็นค่าเริ่มต้น
