# สรุปข้อมูลบนเชน

## 5.9 สรุปข้อมูลบนเชน

สิ่งที่ขึ้นบนเชน ตามหมวดหมู่:

| บนเชน | นอกเชน |
|---|---|
| เหตุการณ์ mint INT | แถว `BintLedgerEntry` แต่ละรายการ |
| เหตุการณ์โอน INT | ยอดคงเหลือ bINT และบันทึกการสะสม |
| รูทการแจกจ่ายต่อรอบ (epoch) | บันทึก ePoints แต่ละรายการ |
| แฮชชุดข้อมูลที่เผยแพร่ (คำมั่นสัญญาความโปร่งใส) | ข้อความดิบ OCR |
| การเปลี่ยนผ่านระดับ NFT | ภาพใบเสร็จและรายการบรรทัด |
| ลายเซ็นธุรกรรมเผา BBB | สัญญาณคะแนนความน่าเชื่อถือ |

กฎคือ: **บนเชนเก็บคำมั่นสัญญาและข้อมูลสรุป นอกเชนเก็บเนื้อหา** ผู้ใช้สามารถตรวจสอบยอดคงเหลือนอกเชนต่อคำมั่นสัญญาบนเชน ในขณะที่เนื้อหาใบเสร็จยังคงอยู่ในชั้นข้อมูลนอกเชน
