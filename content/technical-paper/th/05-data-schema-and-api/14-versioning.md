# การจัดเวอร์ชัน

## 5.13 การจัดเวอร์ชัน

- **เวอร์ชันสคีมา** ปฏิบัติตาม SemVer ระดับบันทึก `schema_version: "1.0.0"` บนทุกใบเสร็จช่วยให้อ่านเข้ากันได้ย้อนหลัง
- **เวอร์ชัน API** มีคำนำหน้า URL (`/v1`, `/v2`) เวอร์ชันหลักสองเวอร์ชันที่ติดกันทำงานคู่ขนานเป็นเวลาอย่างน้อย 12 เดือน
- **เวอร์ชัน taxonomy** เป็นอิสระจากเวอร์ชันสคีมา สินค้ามาตรฐานสามารถย้ายจาก `food.dairy.milk` (v1.0) ไปยัง `food.dairy.fluid-milk` (v1.1) ในขณะที่รูปแบบบันทึกใบเสร็จยังคงเสถียร
