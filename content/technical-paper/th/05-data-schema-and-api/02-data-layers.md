# ชั้นข้อมูล (สรุป Vision)

## 5.1 ชั้นข้อมูล (สรุป Vision)

Vision Paper กำหนดสี่ชั้นข้อมูล ส่วนนี้จับคู่แต่ละชั้นกับที่เก็บข้อมูลเฉพาะ และแสดงว่าสามารถสืบค้นอะไรจากอะไรได้

| ชั้น | สิ่งที่อยู่ที่นี่ | การเข้าถึงผู้ใช้ | การเข้าถึงปฏิบัติการ | การเข้าถึง B2B |
|---|---|---|---|---|
| **อุปกรณ์** | ภาพใบเสร็จต้นฉบับ | ข้อมูลของตน | ขอบเขตอุปกรณ์ | ขอบเขตอุปกรณ์ |
| **ระบบร้อน** | บันทึกใบเสร็จ รายการ 90 วันล่าสุด | ข้อมูลของตน | ปฏิบัติการ | ชั้นข้อมูลสรุป |
| **ระบบอุ่น** | เหมือนระบบร้อน 91 วัน-3 ปี | ข้อมูลของตน | ปฏิบัติการ | ชั้นข้อมูลสรุป |
| **ข้อมูลสรุปที่ไม่ระบุตัวตน** | แผงข้อมูลและดัชนี k-anonymous | มุมมองสรุป | ปฏิบัติการ | มุมมองสรุป |
| **สรุปบนเชน** | แฮชเครดิต bINT เหตุการณ์ INT ระดับ NFT | สาธารณะ | สาธารณะ | สาธารณะ |

กฎที่เคร่งครัด: **ข้อมูลสรุปที่ไม่ระบุตัวตนถูกแยกจากบันทึกผู้ใช้รายเดียว** 5.8 ระบุการแปลง
