# Уровни времени выполнения

## 1.5 Уровни времени выполнения

Yumo Yumo имеет три уровня времени выполнения, каждый с разным бюджетом.

| Уровень | Что здесь выполняется | Бюджет задержки | Бюджет стоимости | Режим отказа |
|---|---|---|---|---|
| Синхронный (запрос) | OCR → LLM → regex → каноническое совпадение → оценка доверия | P95 < ~5 с сквозная | < 0,02 $ / чек | Показать деградированный предпросмотр + повтор |
| Асинхронный (расчёт) | Чеканка bINT, обновления уровня NFT | < 1 часа с момента проверки | Комиссии Solana | Отложить на следующий пакет |
| Ежедневный пакет | Перекластеризация канонических данных, пересчёт оценки здоровья, анонимизированный экспорт, очередь BBB | Ночное окно | Бюджет вычислительного пула | Работа со вчерашним снимком |

Разделение этих уровней позволяет пользовательскому опыту оставаться быстрым, даже когда нижележащие системы дороги или медленны. Стоимость чеканки в Solana (узкое место для субсекундного UX в блокчейне) амортизируется по пакетам.
