# 04 — Токеномические механики

В данном разделе рассматриваются экономические механики протокола Yumo Yumo: роли INT, bINT, ePoints и Foundation NFT; жизненный цикл эмиссии и конверсии; потолки вознаграждений; стейкинг, казначейство и механики выкупа/сжигания; распределение предложения; проекции оборотного предложения; и утилитарность INT.

Юридические заключения по конкретным юрисдикциям управляются на юридическом уровне.

## 4.0 Механическая поверхность

| Механика | Публичный охват |
|---|---|
| INT | Основной on-chain-токен и учёт обращения |
| bINT | Представление вознаграждения до расчёта для верифицированного вклада |
| ePoints | Внутрипродуктовый слой прогресса и сезонной активности |
| Foundation NFT | Представление раннего вклада и уровня |
| Staking | Семантика блокировки, начисления и участия в протоколе |
| Treasury | Взаимосвязь между притоками доходов, операциями, резервами и токенной экономикой |
| Предложение и распределение | Общее предложение, доли рельсов, объёмы токенов (4.16–4.18) |
| Эмиссия и разблокировки | Ежедневные эмиссионные диапазоны на основе MAU, графики разблокировки рельсов (4.19–4.21) |
| Потолки вознаграждений | Потолки на чек и на день по уровню пользователя (4.22–4.23) |
| Конверсия и обращение | Фиксированный коэффициент bINT→INT 1:1, модель оборотного предложения (4.24–4.26) |
| Утилитарность INT | Вознаграждение, стейкинг, выкуп-и-сжигание, управление, связанный доступ (4.27–4.29) |
