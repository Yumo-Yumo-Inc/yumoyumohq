# Версионирование

## 5.13 Версионирование

- **Версии схем** следуют SemVer на уровне записи. `schema_version: "1.0.0"` на каждом чеке позволяет обратно совместимые чтения.
- **Версии API** префиксированы в URL (`/v1`, `/v2`). Две соседние основные версии работают параллельно минимум 12 месяцев.
- **Версии таксономии** независимы от версий схем. Канонический продукт может перемещаться из `food.dairy.milk` (v1.0) в `food.dairy.fluid-milk` (v1.1), в то время как схема записи чека остаётся стабильной.
