# Наблюдение цены (нормативное)

## 5.6 Наблюдение цены (нормативное)

Таблица ценовой памяти. Одна строка на `(canonical_product_id, merchant_id, метка времени наблюдения)`.

```json
// PriceObservation
{
  "observation_id": "01HXY...",
  "canonical_product_id": "cp.pinar.milk.1l",
  "merchant_id": "01HXY...",
  "chain_id": "chain.migros",
  "city": "Istanbul",
  "observed_at": "2026-05-17T14:23:11Z",
  "unit_price_minor": 2350,
  "currency": "TRY",
  "trust_score": "0.XX",
  "is_promotional": false
}
```

Это таблица, которая обеспечивает:

1. **Ценовая память пользователя** — «вы заплатили 23,50 TL за Pınar süt в Migros; медиана на этой неделе — 22,10 TL».
2. **B2B ценовой индекс** — агрегированный по `(canonical_product_id, регион, неделя)` с соблюдением порога k-анонимности.
3. **Импульс инфляции** — высокочастотный индекс корзины, вычисляемый ежедневно.
