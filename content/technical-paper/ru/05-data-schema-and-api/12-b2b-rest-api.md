# B2B REST API

## 5.11 B2B REST API

Отдельная база, отдельная авторизация, отдельные ограничения скорости.

```
Base: https://b2b-api.yumo.io/v1
Auth: API key + аутентифицированные запросы с защитой от повтора. Схема подписи и окно повтора управляются во внутреннем операционном слое.
Rate limit: зависит от тира · отдельная квота от публичного API
```

| Метод | Путь | Назначение |
|---|---|---|
| GET | `/inflation-pulse` | Серия Импульса инфляции TR |
| GET | `/basket-panel` | Запрос Панели корзин |
| GET | `/merchant-benchmarks` | Бенчмарки продавцов |
| POST | `/cohort-query` | Пользовательский когортный запрос с соблюдением k-порога |
| GET | `/catalog` | Доступные продукты + свежесть + ценообразование |
| GET | `/methodology/{version}` | Документ методологии для данной версии |

Каждый B2B-ответ включает `methodology_version`, `k_anonymity_floor` и количество участников ответа, чтобы команда compliance покупателя могла проверить выпуск.
