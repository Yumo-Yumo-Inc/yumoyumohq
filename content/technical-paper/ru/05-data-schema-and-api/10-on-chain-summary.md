# Сводка данных в блокчейне

## 5.9 Сводка данных в блокчейне

Что размещается в блокчейне, по категориям:

| В блокчейне | Вне блокчейна |
|---|---|
| События mint INT | Отдельные строки `BintLedgerEntry` |
| События transfer INT | Балансы bINT и записи начислений |
| Корень распределения за эпоху | Отдельные записи ePoints |
| Хэш опубликованного набора данных (обязательство прозрачности) | Сырой текст OCR |
| Переходы уровней NFT | Изображения чеков и строки чека |
| Подписи TX сжигания BBB | Сигналы оценки доверия |

Правило: **в блокчейне хранятся обязательства и агрегаты; вне блокчейна хранится содержимое.** Пользователь может проверить свой внеблокчейн-баланс по обязательству в блокчейне, при этом содержимое чеков остаётся во внеблокчейн-слое данных.
