# Токены и on-chain-активы

## Токены и on-chain-активы

- **INT** — Передаваемый utility-токен Yumo Yumo, выпущенный на Solana как SPL-токен (decimals = 6, total supply = 99,000,000,000). *См.: 04 Токеномические механики.*
- **bINT** — Soulbound-кредит вклада, mint-ированный на замороженный Associated Token Account (ATA) пользователя при верификации чека. Конвертируется в INT по годовой индексированной ставке. *См.: 04 Токеномические механики.*
- **ePoints** — Деноминированный в USD, soulbound-кредит инсайтов, отражающий «скрытые издержки», выявленные для домохозяйства со временем. *См.: 04 Токеномические механики, 05 Схема данных и API.*
- **cPoints** — Очки вклада до TGE; мигрированы в bINT на событии генерации токенов и сохранены как архивная запись очков. *См.: 04 Токеномические механики.*
- **Foundation NFT (Yumbie)** — NonTransferable NFT стандарта Token-2022, представляющий идентичность пользователя Yumbie. Эволюционирует в Smart Agent на уровне 30. *См.: 00 Команда и философия разработки, 04 Токеномические механики.*
- **Smart Agent** — Пост-30-уровневая эволюция Foundation NFT. Однонаправленное событие mint. *См.: 04 Токеномические механики.*
- **SPL Token** — Стандарт токенов Solana Program Library. «ERC-20 Solana.» Используется для INT.
- **Token-2022** — Стандарт токенов Solana с расширениями. Используется для Foundation NFT с расширением NonTransferable.
- **Frozen ATA** — Associated Token Account, который хранит bINT (или ePoints) и привязан к пользователю mint-авторитетом. Обеспечивает soulbound-поведение на уровне протокола. *См.: 04 Токеномические механики.*
- **Soulbound** — Поведение токена, привязанное к одному кошельку на протяжении жизни актива.
