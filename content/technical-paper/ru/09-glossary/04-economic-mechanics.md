# Экономические механики

## Экономические механики

- **TGE (Token Generation Event)** — Момент, когда INT становится живым on-chain-активом, а cPoints мигрируют в bINT.
- **Эмиссионная кривая** — График эмиссии вознаграждений пользователей, индексированный по ежемесячным активным пользователям и времени с момента TGE. *См.: 04.*
- **Конверсионное соотношение (bINT → INT)** — Годовая индексированная ставка, определяющая, сколько INT получается за единицу bINT. *См.: 04.*
- **Ежедневный потолок bINT** — Максимальное количество bINT, которое пользователь может заработать в день. Вычисляется как `base_cap × level_multiplier × health_score`. *См.: 03 Уровень доверия, 04 Токеномические механики.*
- **Buy-back-and-burn (BBB)** — Механизм сжигания на основе правил, используемый в рамках казначейской политики для сокращения обращения INT. Детали исполнения операционные. *См.: 04.*
- **Liquidity Bootstrapping Pool (LBP)** — Формат DEX-пула для запуска новых токенов с регулируемыми весами. Рассматривается для начальной ликвидности INT.
- **Single-sided LP** — Предоставление ликвидности, требующее только один из двух активов пары. Используется для начального пула INT с заблокированной LP.
