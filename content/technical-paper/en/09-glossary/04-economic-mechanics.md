# Economic mechanics

## Economic mechanics

- **TGE (Token Generation Event)** — The moment INT becomes a live, on-chain asset and cPoints are migrated to bINT.
- **Emission curve** — The user-reward emission schedule, indexed by Monthly Active Users and time since TGE. *See: 04.*
- **Conversion ratio (bINT → INT)** — The year-indexed multiplier that determines how many INT a unit of bINT converts into. *See: 04.*
- **Daily bINT ceiling** — The maximum bINT a user can earn per day. Computed as `base_cap × level_multiplier × health_score`. *See: 03 Trust Layer, 04 Tokenomics Mechanics.*
- **Buy-back-and-burn (BBB)** — Rule-based burn mechanism used under treasury policy to reduce INT circulation. Execution detail is operational. *See: 04.*
- **Liquidity Bootstrapping Pool (LBP)** — A DEX pool format used to launch new tokens with adjustable weights. Considered for initial INT liquidity.
- **Single-sided LP** — Liquidity provision that requires only one of the two assets in a pair. Used to seed the initial INT pool with locked LP.
