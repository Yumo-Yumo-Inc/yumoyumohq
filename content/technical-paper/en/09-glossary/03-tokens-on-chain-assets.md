# Tokens and on-chain assets

## Tokens and on-chain assets

- **INT** — Yumo Yumo's transferable utility token, issued on Solana as an SPL token (decimals = 6, total supply = 99,000,000,000). *See: 04 Tokenomics Mechanics.*
- **bINT** — Soulbound contribution credit minted to a user's frozen Associated Token Account (ATA) when a receipt is verified. Converts to INT on a year-indexed ratio. *See: 04 Tokenomics Mechanics.*
- **ePoints** — USD-denominated, soulbound insight credit reflecting the "hidden cost" surfaced for a household over time. *See: 04 Tokenomics Mechanics, 05 Data Schema and API.*
- **cPoints** — Pre-TGE contribution points; migrated to bINT at Token Generation Event and kept as an archived points record. *See: 04 Tokenomics Mechanics.*
- **Foundation NFT (Yumbie)** — A Token-2022 NonTransferable NFT representing the user's Yumbie identity. Evolves into the Smart Agent at Level 30. *See: 00 Team & Build Philosophy, 04 Tokenomics Mechanics.*
- **Smart Agent** — The post-Level-30 evolution of the Foundation NFT. A one-way mint event. *See: 04 Tokenomics Mechanics.*
- **SPL Token** — Solana Program Library token standard. The "ERC-20 of Solana." Used for INT.
- **Token-2022** — Solana's extension-enabled token standard. Used for Foundation NFTs with the NonTransferable extension.
- **Frozen ATA** — An Associated Token Account that holds bINT (or ePoints) and is bound to the user by the mint authority. Enforces soulbound behaviour at the protocol level. *See: 04 Tokenomics Mechanics.*
- **Soulbound** — Token behaviour bound to a single wallet for the lifetime of the asset.
