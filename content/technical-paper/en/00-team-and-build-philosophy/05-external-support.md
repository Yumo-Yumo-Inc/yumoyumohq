# External support

## 0.3 External support

Outside of the core team, Yumo Yumo's execution is supported by specialist contractors for design, smart-contract audits, security review, and select engineering surfaces. Contractors are engaged through standard service agreements and token distributions are managed through the Proof of Contribution rail. A formal advisory structure with publicly named advisors will be announced before the Token Generation Event.
