# Whitepaper Técnico de Yumo Yumo

El Documento Técnico de Yumo Yumo es la especificación arquitectónica y mecánica del protocolo. El Vision Paper contiene la intención de mercado y la narrativa económica; este documento explica cómo funcionan el procesamiento de recibos, la capa de confianza, la contabilidad de recompensas, el esquema de datos y la liquidación on-chain.

El documento público contiene los contratos del protocolo, los límites de datos, las formas del esquema y las relaciones matemáticas. Los umbrales operativos, la política de enrutamiento de proveedores, los runbooks de defensa y los parámetros de seguridad se gestionan en la capa de operaciones interna.

## Alcance

- Arquitectura del sistema y límites de datos.
- Contratos de etapa para el pipeline de recibos.
- La superficie de decisión pública del modelo de confianza y antiabuso.
- El modelo de trabajo de la mecánica tokenómica y la contabilidad de recompensas.
- Esquema de datos, superficie de API y capa de datos agregados.
- Clases de riesgo y mitigaciones explicables a nivel público.
