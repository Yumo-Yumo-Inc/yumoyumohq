# Zincir üstü veri özeti

## 5.9 Zincir üstü veri özeti

Zincir üstüne ne çıkar, kategoriye göre:

| Zincir üstü | Zincir dışı |
|---|---|
| INT mint olayları | Bireysel `BintLedgerEntry` satırları |
| INT transfer olayları | bINT bakiyeleri ve birikim kayıtları |
| Epoch başına dağıtım kökü | ePoints bireysel kayıtlar |
| Yayınlanmış veri kümesi hash'i (şeffaflık taahhüdü) | OCR ham metin |
| NFT seviye geçişleri | Fiş görselleri ve kalemler |
| BBB yakım TX imzaları | Güven puanı sinyalleri |

Kural: **zincir üstü taahhütleri ve toplamları saklar; zincir dışı içeriği saklar.** Bir kullanıcı, fiş içeriğini açığa çıkarmadan zincir dışı bakiyesini zincir üstü taahhüde karşı doğrulayabilir.
